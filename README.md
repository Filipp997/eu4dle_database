This repository contains the databases used by eu4dle.com — including countries, national ideas, modifiers, and other EU4-related datasets.

The goal is to make the data easy to reuse for tooling, analysis, and community projects.
If you build something with it, feel free to share it — attribution to eu4dle.com is appreciated and bug reports are welcome.

Data extracted/derived from Europa Universalis IV game files. Europa Universalis IV and all related content are property of Paradox Interactive. This repository is fan-made and non-commercial.
